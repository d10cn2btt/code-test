Bootstrap
Datetimepicker JA

$('div.datetimepicker').datetimepicker_ja();

Example1

カスタマイズ版のデフォルトの動作は日付ピッカー

  • 日付ピッカーとして動作
  • 日付選択時にダイアログが閉じる
  • 日本語表示
$('#datetimepicker1').datetimepicker_ja();

Example2

カスタマイズ版を時刻ピッカーとして起動する

$('#datetimepicker2').datetimepicker_ja({
	format : 'hh:mm',
	pickDate : false,
	pickTime : true
});

Example3

分の選択一覧の分間隔を指定する（例では１０分間隔）

$('#datetimepicker3').datetimepicker_ja({
	format : 'hh:mm',
	pickDate : false,
	pickTime : true,
	minutesRange : 10,
	minutesX : 3
});

Example4

オリジナル版の挙動に影響なし

$('#datetimepicker4').datetimepicker({
	format : 'yyyy/MM/dd hh:mm:ss'
});
